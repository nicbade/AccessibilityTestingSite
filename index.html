Accordion Pattern

An accordion is a vertically stacked set of interactive headings that each contain a title, content snippet, or thumbnail representing a section of content.

Accordion Accessibility Features

  • Tab / Shift+Tab: Move focus between accordion headers.
  • Enter or Space: Toggle (expand/collapse) the focused accordion panel.
  • Focus should stay on the triggering elment tht expands/collapses the content, state of the triggering element should be announced to screen reader users.
  • Accordion content that is visually hidden should be properly hidden from screen reader users.

Breadcrumb Accessibility Features

  • Navigation element with the aria-label="Breadcrumb" announces to screen readers that this navigation is breadcrumbs.
  • ol list → communicates the order/hierarchy.
  • Last page is unlinked to indicate it is the current page
  • Keyboard support → focus indicators and hover states are visible.
